# Todo List

A simple client-side Todo List web app built with HTML, CSS and vanilla JavaScript. Add, complete, filter and delete tasks with data persisted in localStorage.

![UI Screenshot](![alt text](image.png))

## Features
- Add tasks
- Mark tasks complete / incomplete
- Filter: All / Active / Completed
- Delete tasks (single or delete all completed via Ctrl+D)
- Task count (items left)
- Persistent storage using localStorage

## Files
- `index.html` — main UI
- `style.css` — styles
- `script.js` — app logic
- `README.md` — this file

## Usage
1. Open `index.html` in your browser.
   - Or serve the folder and open http://localhost:5500
   - Example (Windows, if Python installed):  
     `python -m http.server 5500` then open `http://localhost:5500`
   - Or use VS Code Live Server extension and click "Go Live".
